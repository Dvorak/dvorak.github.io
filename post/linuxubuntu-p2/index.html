⌨️ Caps lock issue on Ubuntu

Remap Caps Lock is by using Gnome Tweak Tool.

  1. You can install it via the Ubuntu Software Center if you wish or by the command line; it does not come installed on Ubuntu by default.
  2. Open a terminal by holding 'CTRL', 'ALT' and 'T' at once or by search it in the dash.
  3. Type in sudo apt-get install gnome-tweak-tool, where apt-get to install gnome-tweak-tool, and sudo to have root permissions to install software.
  4. Open Tweak Tool
  5. Select Keyboard & Mouse on the left bar.
  6. Select Additional Layout Options and then Caps Lock behaviour.
