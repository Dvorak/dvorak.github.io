Least Squares Method - P1

2020-05-27

üìù A Common Technique in State Estimation

Any real-world measurements by the sensor will be imprecise. To get (compute) the physical quantity (like vehicle position) from a set of real-world measurements, the Least Squares Method is used to find the best, or the optimal value is given some assumptions about the sensors and external world.

The method of least squares and also called the ** Squared Error Criterion**. The squared error is the sum of squared errors between what man observed (value of measurements) and what man expects (the best estimate).

Generally,

y: Measured values (m times, y_1 ... y_m)
x: "Actual" value, which is not "exist"
e: Error or measurement noise term

For thIis simplest measurement model with total m times measurement:

y_1 = x + e_1
y_2 = x + e_2
...
y_m = x + e_m

and

e_1 = y_1 - x
e_2 = y_2 - x
...
e_m = y_m - x

The squared error criterion of the case above will be:

e_1^2 + e_2^2 + ... + e_m^2
= (y_1 - x )^2 + (y_1 - x )^2 +... + (y_1 - x )^2 = L(x)

As the y_1 ... y_m are known, here L(x) is the function of x, called Lost Function.

The Least Squares Method could be applied for this case as follows:
The best estimate of the measured values (m times) is the one that minimizes the sum of squared error.

Let's complete the previous definition as follows

y_i: Measured values (m times, y_1 ... y_m)
x: "Actual" value, which is not exist
e_i: Error or measurement noise term
e_i^2: Squared error
x_LS: the best estimate of x (from a set of measurements)
LS(x): Lost function

The state estimate problem now becomes a mathematical problem that finds the x_LS at which the Lost function LS(x) is minimized. This "Minimizing the Squared Error Criterion" in mathematical language will be

x_LS = ArgMin (LS(x), x)

To minimize LS(x), just compute the partial derivative to x, set to 0, and solve for an extremum.

More generally, re-write the criterion using vectors and introduce the Matrix H (called Jacobian Matrix):

e = y - Hx

in which e and y are mx1 dimension vector (m tims measurement), x can be nx1 dimension vector. (estimate in total n states, more than one state). So H has dimensions m x n, where m is the number of measurements and n is the number of stats or parameters that we wish to estimate from measurements.

The H transpose H will be n by n according to the above definition. The matrix will be invertible if and only if (m > = n) m is greater than or equal to n . So, we need at least as many measurements as unknowns to derive the Least Squares Solution for estimation.

Back to the simplest case mentioned previously, it could be derivated that for one unknown state (n=1) and the linear model y = x + e ( all 1 in Jacobian Matrix H), the Least Squares Solution is just the mean of the sum of m times measurements.

x_LS = 1/m (y_1 + y_2 + ... + y_m).

In the discussion above, two assumptions are made regarding using the Least Squares Method to estimate state:

  • Measurement model is linear
  • Measurements are weighted equally (not suspect that the noise are different in measurements, some could have more noise than others)

Q & A

  1. State estimation and parameter estimation?
  • Both are estimation problems. The parameter is constant over time, such as the resistance of a resistor, while the state is defined as a physical quantity that changes over time, such as a vehicle position. Both parameter estimation and state estimation problems belong to the same categories shared similar methods, such as the Least Squares Method.
  1. Real-world usage of state estimation?
  • Localization problem (estimation of position and orientation) in autonomous driving
